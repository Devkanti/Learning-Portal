# 🎓 Learning Management System (LMS)

A full-stack Learning Management System (LMS) designed to deliver a seamless educational experience. This platform features a modern, responsive user interface built with React and Vite, powered by a robust Node.js and MongoDB backend.

## 💻 Tech Stack

**Frontend:**
* React.js
* Vite (for blazing-fast development)
* CSS / JSX

**Backend:**
* Node.js
* Express.js
* MongoDB (Mongoose)

## 📁 Project Structure

This repository is structured as a monorepo containing both the frontend and backend applications:

* `/Frontend` - Contains the Vite/React application.
* `/Backend` - Contains the Node.js API and database configuration.
